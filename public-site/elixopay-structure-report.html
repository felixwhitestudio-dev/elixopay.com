2 รูปแบบธุรกิจ (Business Model)

Elixopay เป็น ระบบรับชำระเงินสำหรับร้านค้า (Merchant Acquiring Service) ที่ช่วยให้ธุรกิจสามารถรับชำระเงินจากลูกค้าผ่านช่องทางออนไลน์ได้อย่างปลอดภัย โดยมีขั้นตอนดังนี้:

⚠️ Elixopay ไม่ได้ให้บริการโอนเงินระหว่างบุคคล (P2P Transfer) — เราให้บริการเฉพาะการรับชำระเงินจากลูกค้าของร้านค้า (Merchant) ที่ผ่านการตรวจสอบ KYC แล้วเท่านั้น

บริการหลัก:

  • Payment Gateway API — รับชำระเงินผ่าน QR PromptPay, บัตรเครดิต/เดบิต, Mobile Banking
  • Merchant Dashboard — ระบบจัดการร้านค้า ดูยอดขาย รายการธุรกรรม
  • Settlement (T+1) — โอนเงินเข้าบัญชีร้านค้าตามรอบที่กำหนด
  • Webhook & API — แจ้งเตือนและเชื่อมต่อระบบอัตโนมัติ
  • KYC Verification — ยืนยันตัวตนร้านค้าก่อนเปิดใช้งาน
3 นโยบายการปฏิบัติตามกฎหมาย (Compliance)

AML/KYC: ดำเนินนโยบายป้องกันการฟอกเงินตาม พ.ร.บ. ป้องกันและปราบปรามการฟอกเงิน พ.ศ. 2542 และแนวทาง FATF

  • Customer Due Diligence (CDD) & Enhanced Due Diligence (EDD)
  • Real-time Transaction Monitoring — ระบบ AI ตรวจจับธุรกรรมผิดปกติ 24/7
  • Suspicious Transaction Reporting (STR) — รายงานธุรกรรมที่น่าสงสัยต่อ ปปง.
  • Sanction Screening — ตรวจสอบรายชื่อ UN, OFAC, EU

ความปลอดภัย: PCI-DSS Level 1 | TLS 1.3 | 3D Secure 2.0 | SOC 2 Type II

ธุรกิจต้องห้าม: การพนันออนไลน์, P2P Money Transfer, Cryptocurrency Exchange, สินค้าผิดกฎหมาย, Ponzi/Pyramid Scheme
