01 ข้อตกลงทั่วไป (General Agreement)

ข้อกำหนดการใช้งานฉบับนี้ ("ข้อกำหนด") เป็นข้อตกลงทางกฎหมายที่มีผลผูกพันระหว่าง ท่าน ("ผู้ใช้บริการ" หรือ "ร้านค้า") กับ บริษัท คัลเจอร์ แคปิตอล เพย์ จำกัด ทะเบียนนิติบุคคลเลขที่ 0105565039818 ซึ่งดำเนินธุรกิจภายใต้ชื่อแบรนด์ Elixopay ("บริษัท" หรือ "เรา")

โดยการเข้าถึง ลงทะเบียน หรือใช้งานบริการใดๆ ของ Elixopay ท่านยืนยันว่า:

  • ท่านได้อ่าน ทำความเข้าใจ และยอมรับข้อกำหนดฉบับนี้โดยสมบูรณ์
  • ท่านมีอำนาจตามกฎหมายในการทำข้อตกลงนี้ (อายุ 20 ปีบริบูรณ์ขึ้นไป หรือเป็นผู้แทนที่ได้รับมอบอำนาจจากนิติบุคคล)
  • ท่านจะปฏิบัติตามกฎหมายและระเบียบข้อบังคับที่เกี่ยวข้องทั้งหมด รวมถึงประกาศของธนาคารแห่งประเทศไทย (BOT)
  • หากท่านไม่ยอมรับข้อกำหนดเหล่านี้ ท่านจะต้องหยุดใช้งานบริการทั้งหมดทันที
สำคัญ: Elixopay เป็น Non-Custodial Payment Gateway (ผู้ให้บริการเทคโนโลยีการชำระเงิน) — เราไม่ได้เก็บรักษา ถือครอง หรือดูแลเงินของลูกค้าร้านค้า เงินที่ลูกค้าชำระจะถูกส่งตรงไปยังบัญชีธนาคารของร้านค้าโดยตรง

02 คำนิยาม (Definitions)

คำศัพท์ต่อไปนี้มีความหมายตามที่กำหนดไว้ด้านล่าง เมื่อใช้ในข้อกำหนดฉบับนี้:

  • "ร้านค้า" (Merchant) — บุคคลธรรมดาหรือนิติบุคคลที่ลงทะเบียนใช้บริการ Elixopay เพื่อรับชำระเงินจากลูกค้า
  • "ลูกค้า" (Customer) — บุคคลที่ทำธุรกรรมการชำระเงินผ่านระบบ Elixopay ให้แก่ร้านค้า
  • "ธุรกรรม" (Transaction) — การทำรายการชำระเงิน การคืนเงิน การยกเลิก หรือการดำเนินการทางการเงินใดๆ ที่ประมวลผลผ่านระบบ Elixopay
  • "การชำระเงิน" (Settlement) — กระบวนการโอนเงินจากธุรกรรมที่สำเร็จไปยังบัญชีธนาคารของร้านค้า หลังหักค่าธรรมเนียมที่เกี่ยวข้อง
  • "API" (Application Programming Interface) — ชุดเครื่องมือและโปรโตคอลที่ Elixopay จัดเตรียมให้ร้านค้าเพื่อเชื่อมต่อระบบการชำระเงินเข้ากับเว็บไซต์หรือแอปพลิเคชันของร้านค้า
  • "Payment Gateway" (ช่องทางชำระเงิน) — ระบบเทคโนโลยีที่ทำหน้าที่เป็นตัวกลางในการประมวลผลและอนุมัติธุรกรรมการชำระเงินระหว่างลูกค้าและร้านค้า
  • "Non-Custodial" (ไม่ถือครองเงิน) — รูปแบบการให้บริการที่ผู้ให้บริการไม่ได้เก็บรักษา ถือครอง หรือดูแลเงินของลูกค้าหรือร้านค้า โดยเงินจะถูกส่งตรงไปยังบัญชีธนาคารของร้านค้าโดยตรง
  • "Chargeback" (การเรียกเงินคืน) — กระบวนการที่ผู้ถือบัตรร้องขอให้ธนาคารผู้ออกบัตรทำการกลับรายการธุรกรรมที่ได้ชำระไปแล้ว
  • "KYC/KYB" — กระบวนการตรวจสอบและยืนยันตัวตนของลูกค้า (Know Your Customer) และธุรกิจ (Know Your Business) ตามที่กฎหมายกำหนด

03 ขอบเขตการให้บริการ (Service Scope)

Elixopay ให้บริการเทคโนโลยีการชำระเงินสำหรับร้านค้าออนไลน์และออฟไลน์ โดยมีขอบเขตการให้บริการดังต่อไปนี้:

  • การประมวลผลการชำระเงิน — รองรับ QR PromptPay, บัตรเครดิต/เดบิต (Visa, Mastercard, JCB), และช่องทางชำระเงินอื่นๆ ที่อาจเพิ่มเติมในอนาคต
  • การ Settlement — โอนเงินจากธุรกรรมที่สำเร็จเข้าสู่บัญชีธนาคารของร้านค้าโดยตรง
  • Merchant Dashboard — แผงควบคุมสำหรับร้านค้าเพื่อจัดการธุรกรรม ดูรายงาน และตั้งค่าระบบ
  • API Access — ชุด API สำหรับเชื่อมต่อระบบการชำระเงินเข้ากับเว็บไซต์หรือแอปพลิเคชันของร้านค้า รวมถึง Webhooks สำหรับการแจ้งเตือนแบบ Real-time
  • Transaction Reporting — ระบบรายงานธุรกรรมแบบละเอียด รวมถึงการส่งออกข้อมูลสำหรับการบัญชีและภาษี
ข้อชี้แจงสำคัญ: Elixopay ไม่ใช่ธนาคาร, ไม่ใช่ผู้ออก e-Money, ไม่ใช่ผู้รับฝากเงิน (Fund Custodian), และไม่ใช่สถาบันการเงินตามที่กำหนดในพระราชบัญญัติธุรกิจสถาบันการเงิน พ.ศ. 2551 — เราเป็นผู้ให้บริการเทคโนโลยีการชำระเงิน (Payment Technology Provider) เท่านั้น ลูกค้าของร้านค้าชำระเงินตรงไปยังบัญชีธนาคารของร้านค้า

บริษัทสงวนสิทธิ์ในการปรับเปลี่ยน เพิ่มเติม หรือยกเลิกบริการบางส่วนได้ โดยจะแจ้งให้ร้านค้าทราบล่วงหน้าตามสมควร

04 การลงทะเบียนและบัญชีผู้ใช้ (Account Registration)

เพื่อใช้บริการ Elixopay ร้านค้าจะต้องลงทะเบียนบัญชีผู้ใช้และดำเนินการตรวจสอบตัวตนตามกระบวนการ KYC/KYB ดังนี้:

  • ข้อมูลที่ต้องจัดส่ง (สำหรับนิติบุคคล): หนังสือรับรองบริษัท, สำเนาบัตรประชาชนกรรมการผู้มีอำนาจ, เอกสารจดทะเบียนภาษีมูลค่าเพิ่ม (ถ้ามี), หนังสือบริคณห์สนธิ, และข้อมูลบัญชีธนาคารของบริษัท
  • ข้อมูลที่ต้องจัดส่ง (สำหรับบุคคลธรรมดา): สำเนาบัตรประชาชน, ทะเบียนการค้า (ถ้ามี), และข้อมูลบัญชีธนาคาร
  • ความถูกต้องของข้อมูล: ท่านรับรองว่าข้อมูลทั้งหมดที่ให้แก่ Elixopay เป็นข้อมูลที่ถูกต้อง ครบถ้วน และเป็นปัจจุบัน ท่านมีหน้าที่อัปเดตข้อมูลเมื่อมีการเปลี่ยนแปลง
  • ความปลอดภัยรหัสผ่าน: ท่านมีหน้าที่รักษาความปลอดภัยของรหัสผ่านและข้อมูลเข้าสู่ระบบ การใช้ Multi-Factor Authentication (MFA) เป็นสิ่งที่แนะนำอย่างยิ่ง
  • ความรับผิดชอบต่อบัญชี: ท่านรับผิดชอบต่อกิจกรรมทั้งหมดที่เกิดขึ้นภายใต้บัญชีของท่าน รวมถึงการใช้งาน API Keys และ Webhooks
  • กระบวนการตรวจสอบ: Elixopay สงวนสิทธิ์ในการตรวจสอบข้อมูลของท่านกับหน่วยงานภาครัฐ สถาบันการเงิน และฐานข้อมูลที่เกี่ยวข้อง เพื่อวัตถุประสงค์ในการป้องกันการฟอกเงินและการทุจริต
บัญชีที่ยังไม่ผ่านกระบวนการ KYC/KYB จะถูกจำกัดการใช้งาน จนกว่าจะส่งเอกสารครบถ้วนและได้รับการอนุมัติจากทีมงาน Elixopay

05 ค่าธรรมเนียม (Fees & Billing)

การใช้บริการ Elixopay มีค่าธรรมเนียมตามที่กำหนดไว้ในแผนราคา (Pricing Plan) ที่ร้านค้าเลือกใช้:

  • ค่าธรรมเนียมธุรกรรม (Transaction Fees): คิดเป็นอัตราร้อยละของมูลค่าธุรกรรม และ/หรือค่าธรรมเนียมคงที่ต่อรายการ ตามที่ระบุไว้ในแผนราคาที่ท่านเลือกใช้
  • รอบการเรียกเก็บเงิน (Billing Cycle): ค่าธรรมเนียมจะถูกหักจากยอดเงินที่ต้อง Settlement ให้ร้านค้าในแต่ละรอบ หรือเรียกเก็บตามรอบบิลที่กำหนด
  • วิธีการชำระเงิน: ร้านค้าสามารถชำระค่าธรรมเนียมผ่านการหักจาก Settlement, การโอนเงิน, หรือวิธีอื่นๆ ที่ Elixopay กำหนด
  • การปรับค่าธรรมเนียม: Elixopay สงวนสิทธิ์ในการปรับเปลี่ยนอัตราค่าธรรมเนียม โดยจะแจ้งให้ร้านค้าทราบล่วงหน้าไม่น้อยกว่า 30 วัน ก่อนมีผลบังคับใช้
  • ไม่มีค่าธรรมเนียมแอบแฝง (No Hidden Fees): ค่าธรรมเนียมทั้งหมดจะถูกแสดงอย่างชัดเจนใน Dashboard และเอกสารสัญญา ไม่มีค่าใช้จ่ายแอบแฝงใดๆ
  • ภาษี: ค่าธรรมเนียมที่แสดงยังไม่รวมภาษีมูลค่าเพิ่ม (VAT) หรือภาษีอื่นๆ ที่อาจเกี่ยวข้อง ร้านค้ามีหน้าที่รับผิดชอบภาษีของตนเอง

06 การชำระเงินและการระงับข้อพิพาท (Payments & Settlement)

Elixopay ดำเนินการ Settlement เงินให้ร้านค้าตามเงื่อนไขดังต่อไปนี้:

  • ระยะเวลา Settlement: ธุรกรรมที่สำเร็จจะได้รับการ Settlement ภายใน T+1 (วันทำการถัดไป) หรือ T+2 (สองวันทำการถัดไป) ขึ้นอยู่กับช่องทางการชำระเงินและแผนบริการ ทั้งนี้ไม่นับรวมวันหยุดราชการและวันหยุดธนาคาร
  • จำนวนเงินขั้นต่ำ (Minimum Payout): การโอนเงินจะดำเนินการเมื่อยอดเงินสะสมถึงจำนวนขั้นต่ำที่กำหนด ตามที่ระบุไว้ในแผนบริการของท่าน
  • การจัดการ Chargeback: ในกรณีที่เกิด Chargeback Elixopay จะแจ้งร้านค้าทราบทันทีพร้อมรายละเอียดธุรกรรมที่เกี่ยวข้อง ร้านค้ามีหน้าที่จัดส่งหลักฐานเพื่อโต้แย้ง (Representment) ภายในระยะเวลาที่กำหนด ค่า Chargeback และค่าธรรมเนียมที่เกี่ยวข้องจะถูกหักจากยอด Settlement ของร้านค้า
  • กระบวนการคืนเงิน (Refund): ร้านค้าสามารถเริ่มต้นกระบวนการคืนเงินผ่าน Dashboard หรือ API ตาม นโยบายการคืนเงิน ของ Elixopay การคืนเงินจะดำเนินการภายใน 5-14 วันทำการขึ้นอยู่กับช่องทางชำระเงิน
  • การระงับข้อพิพาท (Dispute Resolution): ข้อพิพาทเกี่ยวกับธุรกรรมจะได้รับการพิจารณาโดยทีม Support ของ Elixopay ร้านค้าสามารถส่งเรื่องร้องเรียนผ่าน support@elixopay.com หรือระบบ Ticket ใน Dashboard
Elixopay สงวนสิทธิ์ในการระงับการ Settlement ชั่วคราวหากตรวจพบธุรกรรมที่น่าสงสัย การฉ้อโกง หรือการละเมิดข้อกำหนดการใช้งาน จนกว่าจะได้รับการตรวจสอบเรียบร้อย

07 กิจกรรมที่ห้าม (Prohibited Activities)

ร้านค้าตกลงว่าจะไม่ใช้บริการ Elixopay เพื่อวัตถุประสงค์ดังต่อไปนี้ (โปรดดูรายละเอียดเพิ่มเติมใน นโยบายการใช้งานที่ยอมรับได้):

  • กิจกรรมผิดกฎหมาย: การใช้บริการเพื่อวัตถุประสงค์ที่ผิดกฎหมายของประเทศไทยหรือกฎหมายระหว่างประเทศ
  • การฟอกเงิน (Anti-Money Laundering): การใช้บริการเพื่อฟอกเงิน สนับสนุนการก่อการร้าย หรือกิจกรรมที่เกี่ยวข้องกับการเงินผิดกฎหมาย ตามพระราชบัญญัติป้องกันและปราบปรามการฟอกเงิน พ.ศ. 2542
  • การทุจริต (Fraud): การทำธุรกรรมปลอม การใช้บัตรเครดิตที่ถูกขโมย การหลอกลวงลูกค้า หรือการ Manipulation ระบบ
  • สินค้าปลอมแปลง: การจำหน่ายสินค้าปลอม สินค้าละเมิดลิขสิทธิ์ เครื่องหมายการค้า หรือทรัพย์สินทางปัญญาของผู้อื่น
  • อุตสาหกรรมที่ถูกจำกัด: ธุรกิจที่อยู่ในรายการอุตสาหกรรมที่ถูกจำกัดตามประกาศของธนาคารแห่งประเทศไทย (BOT) รวมถึงแต่ไม่จำกัดเพียง: การพนันออนไลน์ที่ผิดกฎหมาย, สินค้าควบคุม, อาวุธ, สารเสพติด
