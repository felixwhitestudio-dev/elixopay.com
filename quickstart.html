คู่มือเริ่มต้นใช้งาน

เริ่มรับการชำระเงินในไม่กี่นาทีด้วย API ที่ทันสมัยของ Elixopay คู่มือนี้จะแนะนำขั้นตอนพื้นฐานในการเชื่อมต่อเกตเวย์การชำระเงินของเราเข้ากับแอปพลิเคชันของคุณ

1. รับ API Keys ของคุณ

ในการสื่อสารกับ Elixopay คุณต้องมี API key สมัครบัญชีฟรีและไปที่ส่วน นักพัฒนา ในแดชบอร์ดของคุณ

เคล็ดลับ คุณจะมีชุดคีย์สองชุด: ชุดหนึ่งสำหรับ โหมดทดสอบ (Test Mode) และอีกชุดสำหรับ โหมดใช้งานจริง (Live Mode)

2. สร้างเซสชันการชำระเงิน

เมื่อลูกค้าพร้อมที่จะชำระเงิน ให้สร้างเซสชันบนแบ็กเอนด์ของคุณ เพื่อให้แน่ใจว่าจำนวนเงินและสกุลเงินจะไม่ถูกดัดแปลง

Request

curl -X POST https://api.elixopay.com/v1/payments \
  -H "Authorization: Bearer sk_test_..." \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 1000,
    "currency": "THB",
    "description": "Order #123456",
    "payment_method_type": "stripe"
  }'

Response

{
  "success": true,
  "data": {
    "id": "pay_123...",
    "checkout_url": "https://elixopay.com/checkout.html?token=...",
    "clientSecret": "pi_..."
  }
}

3. เปลี่ยนเส้นทางไปยังหน้า Checkout

เปลี่ยนเส้นทางลูกค้าของคุณไปยัง checkout_url ที่ได้รับในการตอบกลับ Elixopay จะโฮสต์หน้าชำระเงินที่ปลอดภัย จัดการการตรวจสอบความถูกต้องและการปฏิบัติตามมาตรฐาน PCI ทั้งหมดให้คุณ 

// Frontend Redirect
window.location.href = response.data.checkout_url;

4. ฟัง Webhooks

ลงทะเบียน webhook endpoint เพื่อรับการอัปเดตแบบเรียลไทม์เมื่อการชำระเงินเสร็จสมบูรณ์

Event: payment_intent.succeeded
