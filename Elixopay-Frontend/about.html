Core Values

ค่านิยมที่ขับเคลื่อนทุกสิ่งที่เราทำ

Mission

ทำให้การรับชำระเงินออนไลน์เป็นเรื่องง่ายสำหรับทุกธุรกิจในประเทศไทย ไม่ว่าจะเป็นร้านค้าขนาดเล็กหรือองค์กรขนาดใหญ่

Vision

เป็นโครงสร้างพื้นฐานด้านการชำระเงินชั้นนำของเอเชียตะวันออกเฉียงใต้ที่ขับเคลื่อนด้วยเทคโนโลยีระดับโลก

Security First

ทุกธุรกรรมเข้ารหัส End-to-End พร้อมมาตรฐาน PCI-DSS Level 1 และ 3D Secure 2.0 เพื่อปกป้องข้อมูลลูกค้า

Speed Matters

ประมวลผลธุรกรรมภายใน 2 วินาที ด้วย Cloud-native infrastructure และ Kubernetes orchestration
