01 คำจำกัดความ

  • "Elixopay" หมายถึง บริการ Payment Gateway ที่ดำเนินการโดย บริษัท เอส.พี.ดีเวลลอปเม้นท์ จำกัด
  • "Merchant" หมายถึง ผู้ใช้บริการที่ลงทะเบียนเพื่อรับชำระเงินผ่านระบบ
  • "ผู้ชำระเงิน" หมายถึง ลูกค้าของ Merchant ที่ทำธุรกรรมชำระเงิน
  • "Dashboard" หมายถึง หน้าควบคุมสำหรับจัดการบัญชีและดูรายงาน
  • "API" หมายถึง Application Programming Interface ที่ Elixopay จัดเตรียมให้

02 การยอมรับเงื่อนไข

การลงทะเบียน สร้างบัญชี หรือใช้บริการ Elixopay ถือว่าคุณยอมรับเงื่อนไขนี้ทั้งหมด

หากไม่ยอมรับ กรุณาหยุดใช้บริการ

เราขอสงวนสิทธิ์ในการแก้ไขเงื่อนไขนี้ โดยจะแจ้งให้ทราบล่วงหน้าอย่างน้อย 30 วัน ผ่านอีเมลหรือประกาศบน Dashboard

03 บัญชีผู้ใช้งาน

การลงทะเบียน

  • Merchant ต้องให้ข้อมูลที่ถูกต้อง ครบถ้วน และเป็นปัจจุบัน
  • ต้องผ่านกระบวนการยืนยันตัวตน (KYC) ก่อนเริ่มรับชำระเงินจริง
  • อายุขั้นต่ำ 18 ปี หรือเป็นนิติบุคคลจดทะเบียนถูกต้อง

ความรับผิดชอบ

  • คุณต้องรักษาความลับของ API Key และรหัสผ่าน
  • คุณรับผิดชอบต่อทุกกิจกรรมที่เกิดขึ้นภายใต้บัญชีของคุณ
  • แจ้งเราทันทีหากพบการใช้งานโดยไม่ได้รับอนุญาต

04 บริการของ Elixopay

เราให้บริการ:

  • การรับชำระเงินผ่าน PromptPay, บัตรเครดิต/เดบิต, Mobile Banking
  • Dashboard สำหรับจัดการธุรกรรมและดูรายงาน
  • REST API สำหรับเชื่อมต่อกับระบบของ Merchant
  • ระบบ Webhook สำหรับแจ้งสถานะธุรกรรมแบบ Real-time
  • ระบบ Payout สำหรับถอนเงินเข้าบัญชี Merchant
Target SLA: Uptime 99.9% — แต่ไม่รับประกันว่าบริการจะไม่หยุดชะงักโดยสิ้นเชิง

05 ข้อจำกัดการใช้งาน

ห้ามใช้ Elixopay สำหรับ:

  • กิจกรรมที่ผิดกฎหมายหรือหลอกลวง
  • การฟอกเงินหรือสนับสนุนการก่อการร้าย
  • สินค้าหรือบริการที่ผิดกฎหมายในประเทศไทย
  • การสร้างธุรกรรมปลอมเพื่อทดสอบ (นอกเหนือจาก Sandbox)
  • การเข้าถึงระบบโดยไม่ได้รับอนุญาต

06 ค่าธรรมเนียม

  • ค่าธรรมเนียมจะเป็นไปตามอัตราที่ตกลงกับ Merchant แต่ละราย
  • ค่าธรรมเนียมจะถูกหักจากยอดธุรกรรมก่อนโอนเข้าบัญชี Merchant
  • เราจะแจ้งให้ทราบล่วงหน้าอย่างน้อย 30 วัน หากมีการเปลี่ยนแปลงอัตราค่าธรรมเนียม

07 ทรัพย์สินทางปัญญา

ทุกอย่างที่เกี่ยวข้องกับ Elixopay รวมถึงโลโก้ ซอฟต์แวร์ API และเอกสาร เป็นทรัพย์สินของ บริษัท เอส.พี.ดีเวลลอปเม้นท์ จำกัด

คุณได้รับสิทธิ์ใช้งานแบบจำกัด (non-exclusive, non-transferable) ในระหว่างที่ใช้บริการเท่านั้น

08 การจำกัดความรับผิดชอบ

  • Elixopay ให้บริการ "ตามสภาพ" (as-is) โดยไม่มีการรับประกันใดๆ
  • เราไม่รับผิดชอบต่อความสูญเสียทางอ้อม รายได้ที่สูญเสีย หรือความเสียหายต่อเนื่อง
  • ความรับผิดชอบสูงสุดจำกัดที่ค่าธรรมเนียมที่ Merchant ชำระในช่วง 12 เดือนที่ผ่านมา

09 การระงับและยกเลิกบริการ

โดย Elixopay

เราอาจระงับหรือยกเลิกบัญชีได้ทันทีหาก:

  • ตรวจพบการทุจริตหรือกิจกรรมต้องสงสัย
  • ละเมิดเงื่อนไขการใช้งาน
  • ได้รับคำสั่งจากหน่วยงานราชการ

โดย Merchant

คุณสามารถยกเลิกบัญชีได้ตลอดเวลาโดยแจ้งล่วงหน้า 30 วัน — ยอดเงินคงเหลือจะถูกโอนหลังหักค่าธรรมเนียมคงค้าง
