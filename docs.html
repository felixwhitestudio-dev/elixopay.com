เอกสาร API

ยินดีต้อนรับสู่ Elixopay API คุณสามารถใช้ API ของเราเพื่อเข้าถึง endpoint ต่างๆ ของ Elixopay ซึ่งจะช่วยให้คุณดึงข้อมูลธุรกรรม จัดการกระเป๋าเงิน และประมวลผลการชำระเงินได้

การยืนยันตัวตน (Authentication)

ยืนยันตัวตนบัญชีของคุณเมื่อใช้งาน API โดยระบุ secret API key ในคำขอ (Request) คุณสามารถจัดการ API keys ของคุณได้ที่ แดชบอร์ดนักพัฒนา

เก็บรักษา Key ของคุณให้ปลอดภัย

API keys ของคุณมีสิทธิ์เข้าถึงสูง ดังนั้นต้องเก็บรักษาให้ปลอดภัย! ห้ามแชร์ secret API keys ในที่สาธารณะ เช่น GitHub หรือในโค้ดฝั่ง Client-side

Authorization Header

ทุกคำขอ API (API requests) ต้องแนบ API key ใน Authorization header ดังนี้:

Authorization: Bearer sk_live_...

สร้างการชำระเงิน (Create a Payment)

ในการสร้างเซสชันการชำระเงินใหม่ ให้ส่งคำขอแบบ POST ไปยัง endpoint การชำระเงิน

POST /api/v1/payments 
curl https://api.elixopay.com/v1/payments \
  -H "Authorization: Bearer sk_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 1000,
    "currency": "thb",
    "description": "Order #1234"
  }'
