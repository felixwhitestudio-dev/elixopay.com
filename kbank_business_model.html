คำอธิบายธุรกิจ

Elixopay เป็นระบบรับชำระเงินออนไลน์ (Payment Gateway) ที่ให้บริการร้านค้าและธุรกิจในการรับชำระเงินจากลูกค้าผ่าน QR Code PromptPay โดย Elixopay ให้บริการในรูปแบบ API ที่ร้านค้าสามารถเชื่อมต่อเข้ากับ Website หรือ Application ของตนได้อย่างง่ายดาย

สินค้าและบริการ

  • Payment API — ระบบ API สำหรับสร้าง QR Code PromptPay เพื่อรับชำระเงินอัตโนมัติ
  • Hosted Checkout Page — หน้าชำระเงินสำเร็จรูปที่พร้อมใช้งาน ร้านค้าเพียงส่งลิงก์ให้ลูกค้า
  • Merchant Dashboard — แดชบอร์ดสำหรับร้านค้าในการติดตามรายการชำระเงิน, ดูรายงาน, และจัดการ API Keys
  • Webhook Notifications — ระบบแจ้งเตือนอัตโนมัติไปยังร้านค้าเมื่อการชำระเงินสำเร็จ
  • Sandbox / Test Mode — โหมดทดสอบสำหรับนักพัฒนาในการทดสอบการเชื่อมต่อก่อนใช้งานจริง

6.2 กลุ่มเป้าหมายและจำนวนผู้ใช้บริการ

กลุ่มเป้าหมาย

  • ร้านค้าออนไลน์ (E-commerce) ขนาดเล็กถึงกลาง
  • ธุรกิจบริการที่ต้องการรับชำระเงินผ่าน QR Code
  • นักพัฒนา (Developers) ที่ต้องการเชื่อมต่อระบบรับชำระเงินเข้ากับแอปพลิเคชัน
  • ธุรกิจ SaaS / Subscription ที่ต้องการระบบเรียกเก็บเงินอัตโนมัติ
จำนวนผู้ใช้บริการ (ประมาณการ): บริษัทฯ อยู่ระหว่างเปิดให้บริการ โดยมีร้านค้าที่สนใจสมัครใช้บริการในช่วงแรกประมาณ 10-30 ราย และคาดว่าจะเติบโตตามการขยายฐานลูกค้าในอนาคต

รายละเอียดแต่ละขั้นตอน

ขั้นตอน รายละเอียด
1. สั่งซื้อสินค้า ลูกค้าเลือกสินค้าและกดสั่งซื้อบน Website/App ของร้านค้า
2. สร้าง Payment ระบบร้านค้าเรียก Elixopay API (POST /api/v1/checkout) พร้อมระบุจำนวนเงิน
3-4. สร้าง QR Code Elixopay เรียก KBank QR API เพื่อสร้าง PromptPay QR Code
5-6. แสดง QR ร้านค้าแสดง QR Code หรือส่ง Checkout Link ให้ลูกค้า
7. สแกนจ่ายเงิน ลูกค้าสแกน QR ด้วยแอป Mobile Banking ใดก็ได้
8. Callback KBank ส่ง Callback แจ้ง Elixopay ว่าชำระเงินสำเร็จ
9. Webhook Elixopay ส่ง Webhook (payment.success) แจ้งร้านค้า
10. ยืนยัน ร้านค้าอัพเดทสถานะคำสั่งซื้อและแจ้งลูกค้า
