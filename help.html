FAQs

ลืมรหัสผ่านทำอย่างไร?

จะมีฟังก์ชัน Reset Password ในเวอร์ชันถัดไป — ขณะนี้สามารถติดต่อทีม Support เพื่อ Reset ให้ชั่วคราว.

ต้องใช้ Test Card อะไรกับ Stripe?

ใช้เลข 4242 4242 4242 4242, วันหมดอายุอนาคต, CVC 3 หลัก, ZIP ใดก็ได้.

Webhook ไม่อัพเดทสถานะ?

ตรวจสอบว่า Stripe CLI ทำงานและสิทธิ์ถูกต้อง (stripe listen) และค่า Webhook Secret อยู่ในไฟล์ .env.

ระบบปลอดภัยแค่ไหน?

ดูสรุปที่หน้า Security หรือไฟล์ SECURITY_README.md — มี JWT, Rate Limit, Account Lockout, Helmet headers และอื่นๆ.
